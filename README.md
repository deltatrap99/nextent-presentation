# 🏕️ NEXTENT — Interactive Presentation

> **Next-Gen Sustainable Modular Tent** | SR12 Team

An immersive, cinematic web presentation for the NEXTENT modular camping tent project. Built for pitch roadshow demos with interactive game mechanics and premium visual design.

## ✨ Features

### 🎬 Cinematic Storytelling Player
- 8-scene auto-playing product narrative (COOL → HEAL → BUILD)
- Ken Burns zoom effects, dot navigation, progress bar
- Auto-start/pause on scroll with IntersectionObserver

### 🎮 30-Second Challenge Game
- Draw festival situation cards & fix your tent
- Interactive SVG tent with highlighted damage zones
- Module status bar, streak tracking, score system
- Premium glassmorphism UI with `nx-game` prefix

### 🎵 Dramatic Countdown Music
- Web Audio API engine with 3-layer sound design
- Low drone + heartbeat pulse + high tension escalation
- BPM increases as time decreases (70→180 BPM)

### 📝 Registration Form
- Premium glassmorphism waitlist form
- Loading spinner, confetti success animation
- Random member number badge

### 🎨 Design System
- **Theme**: Dark forest green, cream, gold, teal
- **Typography**: Playfair Display + Outfit (Google Fonts)
- **Effects**: Glassmorphism, particle system, scroll reveals
- **Responsive**: Mobile-first breakpoints

## 🚀 Quick Start

```bash
# Serve with any static server
npx serve .

# Or Python
python -m http.server 8090
```

Then open `http://localhost:8090/build-nextent.html`
